# Devcamper API

> Backend API for Devcamper application, which is bootcamp directory website

## Usage
Rename "config.env.env" to "config.env" and update the values/settings to your own

## Install Dependencies
```
npm install
```

## Run App
```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
